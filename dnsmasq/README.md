#### Volume:

- /etc/dnsmasq.d

#### Custom usage:

    docker run \
        -d \
        --name dnsmasq \
        -p 53:53/tcp \
        -p 53:53/udp \
        -v ./dnsmasq.d:/etc/dnsmasq.d \
        xutongle/dnsmasq

#### Compose example:

    dnsmasq:
      image: xutongle/dnsmasq
      ports:
        - "53:53/tcp"
        - "53:53/udp"
      volumes:
        - ./dnsmasq.d:/etc/dnsmasq.d
      restart: always
